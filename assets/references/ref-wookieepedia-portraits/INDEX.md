# Wookieepedia Character Portraits

60 character portrait images from Wookieepedia (starwars.fandom.com) for all playable characters in Star Wars Rebellion.

**Purpose**: HD reference images for Gemini-based upscaling of the original 80x80 GOKRES.DLL portraits. Fed as style references alongside the source BMP to guide detail reconstruction.

**Source**: Wookieepedia MediaWiki API (`pageimages` property, 500px thumbnails)
