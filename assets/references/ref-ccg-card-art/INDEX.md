# Star Wars CCG Card Art

56 character card images from the Star Wars Customizable Card Game (Decipher) and related TCG sources via Wookieepedia.

**Purpose**: HD painted character portraits for Gemini multi-image upscaling of GOKRES.DLL 80x80 portraits. The CCG art is painted in the same era (1995-2001) and depicts the exact same characters.

**Source**: Wookieepedia MediaWiki API — file namespace search for character CCG images.

**Naming**: `{faction}-{name-slugified}-ccg.jpg`

## Coverage
- 29 Alliance characters
- 27 Empire characters
- Painted art style matches 1998 Rebellion aesthetic
